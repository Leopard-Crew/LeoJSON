LeoJSONReadOptions

Options controlling JSON parsing. 

typedef enum { 
    LeoJSONReadStrict = 0 
} LeoJSONReadOptions;
Constants
LeoJSONReadStrict
Strict JSON parsing. This is currently the only supported read option.

LeoJSONWriteOptions

Options controlling JSON serialization. 

typedef enum { 
    LeoJSONWriteCompact = 0 
} LeoJSONWriteOptions;
Constants
LeoJSONWriteCompact
Compact JSON output. This is currently the only supported write option.
