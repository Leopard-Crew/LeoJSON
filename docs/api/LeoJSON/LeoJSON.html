LeoJSON

Leopard-Crew JSON boundary for Mac OS X 10.5.8 PowerPC.


LeoJSON provides the public JSON API boundary for Leopard-Crew projects.

JSONKit remains an internal parser and serializer engine. Application code should include LeoJSON.h and must not include JSONKit.h directly.

LeoJSON follows Cocoa naming and ownership conventions. Returned objects are autoreleased unless otherwise documented.
