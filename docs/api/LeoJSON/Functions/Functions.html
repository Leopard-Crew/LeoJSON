LeoJSONDataFromObject

Serializes a Foundation object graph to compact UTF-8 JSON data. 

NSData *LeoJSONDataFromObject(
    id object, 
    LeoJSONWriteOptions options, 
    NSString **errorString);
Parameters
object
The Foundation object graph to serialize. Must not be nil.
options
JSON write options. Currently only LeoJSONWriteCompact is supported.
errorString
Optional output pointer receiving an autoreleased diagnostic string on failure. May be NULL.
Return Value

Autoreleased JSON data, or nil on failure.

Discussion

The caller does not own the returned data. Retain or copy it if it must live beyond the current autorelease pool.

The supported Foundation object graph follows LeoJSON's current API contract and the vendored JSONKit engine behind the LeoJSON boundary.

LeoJSONObjectFromData

Parses UTF-8 JSON data into Foundation objects. 

id LeoJSONObjectFromData(
    NSData *data, 
    LeoJSONReadOptions options, 
    NSString **errorString);
Parameters
data
The JSON data to parse. Must not be nil.
options
JSON read options. Currently only LeoJSONReadStrict is supported.
errorString
Optional output pointer receiving an autoreleased diagnostic string on failure. May be NULL.
Return Value

An autoreleased Foundation object graph, or nil on failure.

Discussion

The caller does not own the returned object. Retain or copy it if it must live beyond the current autorelease pool.

If parsing fails and errorString is not NULL, LeoJSON stores an autoreleased diagnostic string in *errorString. Diagnostic strings are intended for logging and developer feedback, not as stable machine-readable error codes.
